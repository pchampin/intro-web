Comment utiliser ce support
+++++++++++++++++++++++++++

Ce support de cours a deux structures superposées :

* son organisation par défaut est celle d'un manuel de référence,
  dans laquelle les contenus sont regroupés par thème
  (`historique <intro>`:doc:, `html`:doc:, `css`:doc:) ;

* il propose aussi `un parcours plus pédagogique <seances>`:doc:,
  qui propose un plan de séances alternant entre des différents thèmes,
  et se termine par un `projet <projet>`:doc:.

Notez aussi que les chapitres peuvent être visualisés sous force de présentation
(option *View as slides* dans le menu de gauche).
